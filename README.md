# HariboteOS
HariboteOS 「30日で出来る！OS自作入門」

![Imgur](https://i.imgur.com/a07oRuE.png)

## ディレクトリ構成

- haribote
  - OS本体

- build_result
  - ビルドされたファイルがコピーされる先のディレクトリ

- mmlfiles
  - MMLファイル

- images
  - 画像ファイル

- その他
  - アプリケーション
